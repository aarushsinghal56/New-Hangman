1. Play Locally

Perfect for playing solo or with someone next to you.

Play Hot-Seat (2 Players): The classic party mode. One player types a secret word, then the other player guesses.

Play vs. Bot (1 Player): Challenge the computer! You can select:

Category: Movies, Animals, Car Brands, or Food.

Difficulty:

Easy: Vowels + 1 consonant revealed. Uses simple words.

Medium: Only vowels revealed. Uses common words.

Hard: Only vowels revealed. Uses more complex and rare words.

2. Play Online with Friends

The full real-time multiplayer experience.

Create/Join: One player (the Host) creates a room. The other joins with the "Game Code".

Set Match: The Host chooses the number of rounds (e.g., Best of 7).

Play: Players take turns being the "Word Setter" and the "Guesser".

Chat: Talk (and send emojis!) to your friend in the chat window.

Win: The first player to win the majority of rounds wins the match!

Rematch: After the match, the Host can start a new match in the same room.
